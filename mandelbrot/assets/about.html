Mandelbrot Map

version 0.5, by Ralf, September 2008.
Visit the Mandelbrot project page.

Summary: Mandelbrot Map is a simple Mandelbrot fractal viewer for Android that acts as an "infinite map". Simply use it like Google Maps, with panning and zooming.

Keys:

  • I : zoom in
  • O : zoom out
  • S : next point of interest ("P.O.I.")
  • G : toggle grid

Enjoy!
