Mandelbrot Map

version 0.6, by Ralf, September 2008.
Visit the Mandelbrot project page.

Mandelbrot Map is a simple Mandelbrot fractal viewer for Android that acts as an "infinite map". Simply use it like Google Maps, with panning and zooming.

Located under Menu > More you will find options to save images and set as wallpaper.

Keys:

  • I : zoom in
  • O : zoom out
  • S : next point of interest ("P.O.I.")
  • C : clear tile cache (for benchmarking)
  • F : toggle fly mode (for benchmarking)

Enjoy!
